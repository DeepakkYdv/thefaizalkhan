# Simple Portfolio Web App

This is a minimal portfolio starter built with React and Vite and configured for Yarn.

## Folder structure

```text
.
|-- index.html
|-- package.json
|-- vite.config.js
|-- src
|   |-- App.jsx
|   |-- main.jsx
|   |-- components
|   |   |-- Contact.jsx
|   |   |-- Header.jsx
|   |   `-- ProjectCard.jsx
|   |-- data
|   |   `-- projects.js
|   `-- styles
|       `-- global.css
`-- README.md
```

## Run with Yarn

```bash
corepack enable
yarn install
yarn dev
```
